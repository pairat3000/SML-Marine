# 🚢 SML-Marine Dashboard

ระบบ Case Management Dashboard สำหรับติดตามและวิเคราะห์เคสของ Marine App & SML App แบบ Unified View

🌐 **Demo:** [https://pairat3000.github.io/SML-Marine/](https://pairat3000.github.io/SML-Marine/)

---

## Features

- **ภาพรวม (Overview)** — สรุปเคสทั้งหมดจากทั้ง 2 แอปในหน้าเดียว
- **Marine App Tab** — ดูเคสเฉพาะของ Marine App
- **SML App Tab** — ดูเคสเฉพาะของ SML App
- **KPI Cards** — แสดงจำนวนเคสแยกตามประเภท (CR, Defect, Issue, รอดำเนินการ, Done)
- **Chart: เคสตาม Module (TOP 10)** — วิเคราะห์ว่า Module ไหนมีปัญหามากที่สุด
- **Chart: ประเภทเคส** — สัดส่วน CR / Defect / Issue
- **Chart: เปรียบเทียบ Marine vs SML** — เทียบปริมาณเคสระหว่าง 2 แอป
- **รายการเคสทั้งหมด** — ค้นหา / กรองตาม CR, Defect, Issue
- **LIVE Mode** — แสดงข้อมูล Real-time

---

## ประเภทเคส

| ประเภท | คำอธิบาย |
|--------|----------|
| CR | Change Request — ขอเพิ่ม/แก้ไข Feature |
| Defect | บัก / ข้อผิดพลาดในระบบ |
| Issue | ปัญหาการใช้งานทั่วไป |
